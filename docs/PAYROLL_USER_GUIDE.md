# Arts & Aging - Payroll Portal User Guide

Welcome to the improved Payroll Portal. This guide outlines the new features and how to use them efficiently.

## 1. Dashboard Overview
The dashboard provides a snapshot of your work status.
*   **Stats Cards**: View pending requests, monthly hours, and *new* weekly hours summary.
*   **Quick Actions**: One-click access to common tasks.
*   **Recent Activity**: A unified feed of your time entries and expense requests.

## 2. Daily Check-in
*   Located directly on the dashboard.
*   Enter your total hours worked for the day (0-24 range).
*   System automatically logs the entry and updates your statistics.

## 3. Managing Requests
Navigate to "My Requests" to manage expenses and time off.
*   **New Feature: Filtering**: Use the dropdowns at the top to filter requests by **Category** (Expense, Sick Day, Time Off) or **Status** (Pending, Approved, Rejected).
*   **Sorting**: Requests are automatically sorted by newest first.

## 4. Keyboard Shortcuts
For power users, we have added global keyboard shortcuts to navigate the portal quickly:
*   `Alt + N`: Go to **New Request** page.
*   `Alt + D`: Return to **Dashboard**.
*   `Alt + P`: Go to **My Profile**.

## 5. Security & Auditing
*   All time entries and request submissions are now automatically logged to the secure system audit trail for compliance.
*   File uploads are validated for type (PDF/Image) and size (max 5MB).

## 6. Support
If you encounter any issues, please contact the system administrator.
